What I've learned so far...

In Mechatronics we cover a vast range of subjects that are included in both the Mechanical and Electrical fields of Engineering. On the Mechanical side of things I've learned thermodynamics, fluid mechanics, material science, strength of materials, machine design and vibration and noise. On the Electrical side I've studied electro-techniques, computer programming, numerical methods, electrical drive systems, control systems and computer systems. Besides from these I had to also learn the more general Engineering subjects like Engineering mathematics, applied mathematics, Engineering chemistry, Engineering physics and drawings! Also due to the numerous group projects, group members and I have learned invaluable team building skills, leadership, problem solving, time management and conflict management.

What can I code

In Engineering we learned C code in computer programming, this was used in computer systems when programming micro-controller boards (STM32) while learning the additonal hardware abstraction layer (HAL) in order to manipulate and programe the board. With this board and an online emulator our project was to recreate the "Space Invader" game. We also consistently used MATLAB and Stimulink for Control systems, Vibration and Noise and Machine Design. I have also learned the basics in autoCAD invertor to design structures and projects. Above this I have very basic coding in G-codes when programming CNC machines as well as pogramming in a Fanuc virtual reality software to control robotic arms. For my own benefit I have learned Excel programming with basic VBA coding as well as Web-Development using HTML, CSS and javaScript while being able to incoperate Bootstrap and JQuery libraries into these sites.
